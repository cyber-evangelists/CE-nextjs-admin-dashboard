<h2>
    Quick Start
</h2>
<ol>
    <li>Make sure you have the latest stable versions for Node.js and NPM installed</li>
    <li>Clone repository: <code>git clone https://github.com/cyber-evangelists/CE-nextjs-admin-dashboard.git</code></li>
